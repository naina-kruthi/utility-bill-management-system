Utility Bill Management System

A feature-rich desktop application built using Java Swing that helps users manage and track utility bills efficiently. This project follows Object-Oriented Programming (OOP) principles and provides a clean, user-friendly interface.

 Features:
 Add and manage utility bills (electricity, water, internet, etc.)
 View bill details and history
 Generate and track payment records
 User-friendly GUI using Java Swing
 Modular architecture with proper OOP design
 Easy data handling and separation of concerns

Technologies Used:
Java
Java Swing (GUI)
OOP Concepts
Encapsulation
Inheritance
Polymorphism
Abstraction
